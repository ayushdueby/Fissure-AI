# Fissure AI Frontend

Production-style frontend for the Fissure AI Git backend.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- ShadCN-style UI components
- Axios
- Framer Motion
- Sonner toasts

## Backend Target

The frontend is wired to:

`http://localhost:8080/git`

## Install

```bash
npm install
```

## Run Frontend

```bash
npm run dev
```

## Build for Production

```bash
npm run build
npm run start
```
